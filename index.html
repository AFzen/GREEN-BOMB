O que é o Green Bomb?

A visível necessidade de cuidar do meio ambiente é fundamental para os jovens.
Com esse produto problemas como erosão em locais inacessíveis são resolvidos, já que o produto permite jogar semente de plantas nesses locais, que é um dos meios de resolver este problema.
